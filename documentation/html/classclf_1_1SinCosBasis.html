Detailed Description

Represent a local function using a sine-cosine basis.

We need to define basis functions \(\{\phi^{(i)}\}_{i=1}^{n}\) for the local function \(\ell: \mathbb{R}^{d} \mapsto \mathbb{R}\).

Given the number \(j \in \mathbb{N}\), define the scalar functions \(l_i: \mathbb{R} \mapsto \mathbb{R}\)

\begin{equation*} \begin{array}{cccccc} l_0(x) = 1 & l_{2j}(x) = \cos{(\pi j x)} & \mbox{(even numbers)} & \mbox{and} & l_{2j-1}(x) = \sin{(\pi j x)} & \mbox{(odd numbers.)} \end{array} \end{equation*}

Let \(\iota = (i_0,\, i_1,\, ...,\, i_d)\) define a multi-index such that the \(i^{th}\) basis function \(\phi^{(i)}: \mathbb{R}^{d} \mapsto \mathbb{R}\) is

\begin{equation*} \phi^{(i(\iota))} = \prod_{j=1}^{d} l_{i_j}(x_j). \end{equation*}

For example, \(\iota = (0,\, 0,\, ...,\, 0)\) defines the constant basis function \(\phi^{(0)}(x) = 1\), \(\iota = (1,\, 0,\, ...,\, 0)\) defines \(\phi^{(i(\iota))}(x) = \sin{(\pi x_1)}\), and \(\iota = (2,\, 0,\, ...,\, 0)\) defines \(\phi^{(i(\iota))}(x) = \cos{(\pi x_1)}\), where we have implicitly defined a mapping between the index \(i\) and the multi-index \(\iota\).

std::shared_ptr< SinCosBasis > SinCosBasis::TotalOrderBasis ( boost::property_tree::ptree const &  pt)
static

Construct a total-order sine/cosine basis.

For a multi-index \(\iota\) define \(\vert \iota \vert = \sum_{i=1}^{d} i_d\). This function constructs a polynomials basis using all of the multi-indices such that \(\vert \iota \vert \leq 2 p\), where the order \(p\) is a user-supplied parameter.

Note we use all of the multi-indices whose order is bounded by \(2 p\) so that we include all of the sine and cosine terms.

Configuration Parameters (in addition to the usual parameters for a PolynomialBasis):

Parameter Key Type Default Value Description
"InputDimension" std::size_t — The input dimension for the basis functions (the dimension \(d\)).
"Order" std::size_t 1 The order of the multi-indices (the parameter \(p\)).
