Detailed Description

template<typename MatrixType>
class clf::QuadraticCostOptimizer< MatrixType >

Minimize a cost function with the form \(J(p) = \| A p - f \|^2\).

Solve the minimization problem

\begin{equation*} \hat{p} = \mbox{arg min}_{p \in \mathbb{R}^{q}} \sum_{i=1}^{n} \|f_i(p)\|^2 \end{equation*}

where all \(f_i\) are a linear functions of the parameter \(p\). The residual given a parameter value \(p_0\) is \(r(p) = f(p_0) + J (p-p_0)\) such that \(f(p_0)\) is computed by clf::CostFunction::CostVector, the matrix \(J\) is computed by clf::CostFunction::Jacobian, and the cost is \(r(p) \cdot r(p)\). Therefore, we solve \(J^{\top} J \delta = - J^{\top} f(p_0)\) and set the optimal parameter value \(\hat{p} = p_0 + \delta\).
