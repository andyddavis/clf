Detailed Description

template<typename MatrixType>
class clf::LevenbergMarquardt< MatrixType >

An implementation of the Levenberg Marquardt algorithm.

Used to minimize cost functions that are children of clf::CostFunction.

Configuration Parameters: Requires all of the configuration parameters in clf::Optimizer and

Parameter Key Type Default Value Description
"GradientTolerance" double 1.0e-10 The tolerance for the gradient norm
"FunctionTolerance" double 1.0e-10 The tolerance for the cost function
"ParameterTolerance" double 1.0e-10 The tolerance for the parameter norm
"MaximumFunctionEvaluations" std::size_t 1000 The maximum number of function evaluations
"MaximumJacobianEvaluations" std::size_t 1000 The maximum number of Jacobian evaluations
"MaximumHessianEvaluations" std::size_t 1000 The maximum number of Hessian evaluations
"MaximumIterations" std::size_t 1000 The maximum number of iterations
"InitialDampling" double 1.0 The value of the damping parameter at the first iteration
"DampingShrinkFactor" double 0.1 The factor multiplying the damping factor if the error has decreased
"DampingGrowFactor" double 2.0 The factor multiplying the damping factor if the error has increased
"LineSearchFactor" double 0.5 Each iteration of the line search parameter decreases the step size by this factor
"MaxLineSearchSteps" std::size_t 10 The maximum number of line search iterations
"GaussNewtonHessian" bool false true: Use the Gauss-Newton Hessian to compute the step direction, false: Use the full Hessian to compute the step direction
