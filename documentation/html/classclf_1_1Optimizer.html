Detailed Description

template<typename MatrixType>
class clf::Optimizer< MatrixType >

A generic optimization algorithm.

Configuration Parameters:

Parameter Key Type Default Value Description
"GradientTolerance" double 1.0e-10 The tolerance for the gradient norm.
"FunctionTolerance" double 1.0e-10 The tolerance for the cost function.
"MaximumFunctionEvaluations" std::size_t 1000 The maximum number of function evaluations.
"LinearSolver" std::string "Cholesky" The linear solver used by clf::SolveLinearSystem; Choose between: "Cholesky", "CholeskyPivot", "LU", or "QR". The solver will default to LU for any invalid input string. See clf::LinearSolver.
