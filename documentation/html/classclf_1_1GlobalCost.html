◆ NumCostFunctions()

std::size_t GlobalCost::NumCostFunctions ( std::shared_ptr< SupportPointCloud > const &  cloud)
staticprivate

Compute the number of sub-cost function.

The cost function takes the form:

\begin{equation*} C = \min_{\boldsymbol{\beta} \in \mathbb{R}^{n}} \sum_{i=1}^{m} f_i(\boldsymbol{\beta})^{2}. \end{equation*}

This function computes \(m\) based on the uncoupled and coupled costs that are associated with each support point.

Parameters
[in]cloudThe support point cloud that holds all of the points (and their local cost functions)
Returns
The number of sub-cost functions
