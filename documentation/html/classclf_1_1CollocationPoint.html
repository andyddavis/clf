◆ localIndex

std::size_t clf::CollocationPoint::localIndex
private

The local index of this collocation point.

Each collocation point is associated with a support point; this support point is the nearest support point. The local index \(k\) indicates that this is the \(k^{th}\) collocation point associated with that particular support point.

The local indices are not ordered in any way.
