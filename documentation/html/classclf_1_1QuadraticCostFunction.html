Detailed Description

template<typename MatrixType>
class clf::QuadraticCostFunction< MatrixType >

A clf::CostFunction with linear penalty functions.

The cost function is the sum of \(m\) squared penalty functions \(f_i: \mathbb{R}^{n} \mapsto \mathbb{R}^{d_i}\),

\begin{equation*} C = \min_{\boldsymbol{\beta} \in \mathbb{R}^{n}} \sum_{i=1}^{m} \| f_i(\boldsymbol{\beta}) \|^{2}. \end{equation*}

In the case of a quadratic cost function the penalty functions \(f_i\) are all linear with respect to \(\beta\), implying that \(f_i(\beta) = A_i \beta - g_i\) for \(A \in \mathbb{R}^{d_i \times n}\) and \(g_i \in \mathbb{R}^{d_i}\). The matrix \(A_i\) as the Jacobian of \(f_i\) and \(g_i\) is the right hand side.
