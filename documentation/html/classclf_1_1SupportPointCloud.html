Detailed Description

The cloud of support points \(\{x^{(i)}\}_{i=1}^{n}\) that defines the coupled local function.

Configuration Parameters:

Parameter Key Type Default Value Description
"MaxLeaf" std::size_t 10 The max leaf parameter for constructing the \(k\)- \(d\) tree.
"RequireConnectedGraphs" bool false true: The graphs associated with each output must be connected, false: The graphs associated with each output need not be connected
