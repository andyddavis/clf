Detailed Description

Compute the uncoupled cost associated with the support point at \(\hat{x}\).

Let \(\Phi_{\hat{x}}(x)\) be the \(m \times \tilde{q}\) matrix that defines the local function \(\ell_{\hat{x}}(x) = \Phi_{\hat{x}}(x) p\) for \(p \in \mathbb{R}^{\tilde{q}}\) (see clf::SupportPoint). The uncoupled cost function associated with the support point is

\begin{equation*} p_{\hat{x}} = \mbox{arg min}_{p \in \mathbb{R}^{\tilde{q}}} J(p) = \sum_{j=0}^{k_{nn}} \frac{m}{2} \| \mathcal{L}_{\hat{x}}(\Phi_{\hat{x}} (x_{I(\hat{x},j)}) p) - f(x_{I(\hat{x},j)}) \|^2 {K_{\hat{x}}(\hat{x}, x_{I(\hat{x},j)})} + \frac{a}{2} \|p\|^2, \end{equation*}

where \(I(\hat{x}, j)\) is the \(j^{th}\) closest point to the support point \(\hat{x}\).

Parameter Key Type Default Value Description
"RegularizationParameter" double 0.0 The regularization parameter \(a\).
"UncoupledScale" double 1.0 The scale parameter \(m\).

Evaluate the \(i^{th}\) penalty function.

For \(i \in [0, k_{nn}]\) where \(k_{nn}\) is the number of nearest neighbors the \(i^{th}\) penalty function is

\begin{equation*} \sqrt{\frac{m}{2} K_i(x_i, x_{I(\hat{x},j)})} \| \mathcal{L}_{\hat{x}}(\Phi_{\hat{x}} (x_{I(\hat{x},j)}) p) - f(x_{I(\hat{x},j)}) \| \end{equation*}

and for \(i=k_{nn}+1\) (only if \(a>0\)) the final penalty function is

\begin{equation*} \sqrt{\frac{a}{2}} \|p\|. \end{equation*}

Evaluate the gradient of the \(i^{th}\) penalty function.

For \(i \in [0, k_{nn}]\) where \(k_{nn}\) is the number of nearest neighbors the \(i^{th}\) penalty function gradient is

\begin{equation*} \sqrt{\frac{m}{2} K_i(x_i, x_{I(\hat{x},j)})} \frac{(\nabla_{p} \mathcal{L}_{\hat{x}}(\Phi_{\hat{x}} (x_{I(\hat{x},j)}) p))^{\top} ( \mathcal{L}_{\hat{x}}(\Phi_{\hat{x}} (x_{I(\hat{x},j)}) p) - f(x_{I(\hat{x},j)}) ) }{ \| \mathcal{L}_{\hat{x}}(\Phi_{\hat{x}} (x_{I(\hat{x},j)}) p) - f(x_{I(\hat{x},j)}) \| } \end{equation*}

and for \(i=k_{nn}+1\) (only if \(a>0\)) the final penalty function gradient is

\begin{equation*} \sqrt{\frac{a}{2}} \frac{p}{\|p\|}. \end{equation*}
