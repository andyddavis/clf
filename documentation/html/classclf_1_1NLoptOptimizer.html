Detailed Description

template<typename MatrixType>
class clf::NLoptOptimizer< MatrixType >

Use NLopt to solve the optimization problem.

Configuration Parameters: Requires all of the configuration parameters in clf::Optimizer and

Parameter Key Type Default Value Description
"GradientTolerance" double 1.0e-10 The tolerance for the gradient norm.
"FunctionTolerance" double 1.0e-10 The tolerance for the cost function.
"MaximumFunctionEvaluations" std::size_t 1000 The maximum number of function evaluations.
"Algorithm" std::string "LBFGS" The NLopt algorithm (see MUQ documentation for a list of options).
