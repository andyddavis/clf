Detailed Description

template<typename MatrixType>
class clf::CostFunction< MatrixType >

A cost function that can be minimized using the Levenberg Marquardt algorithm.

The cost function is the sum of \(m\) squared penalty functions \(f_i: \mathbb{R}^{n} \mapsto \mathbb{R}^{d_i}\),

\begin{equation*} C = \min_{\beta \in \mathbb{R}^{n}} \sum_{i=1}^{m} \| f_i(\beta) \|^{2}. \end{equation*}

Compute the Hessian of the cost function.

The Hessian of the cost function is

\begin{equation*} H = 2 \sum_{i=1}^{m} \left( \sum_{j=1}^{d_i} f_i^{(j)}(\beta) \nabla_{\beta}^2 f_i^{(j)}(\beta) + (\nabla_{\beta} f_i(\beta))^{\top} \nabla_{\beta} f_i(\beta) \right), \end{equation*}

where \(\nabla_{\beta}^2 f_i^{(j)}(\beta)\) is the Hessian of the \(j^{th}\) output of \(f_i\). Alternatively, we could compute the Gauss-Newton approximation

\begin{equation*} H = 2 \sum_{i=1}^{m} (\nabla_{\beta} f_i(\beta))^{\top} \nabla_{\beta} f_i(\beta). \end{equation*}

Parameters
[in]betaThe current parameter value
[out]hessThe Hessian matrix
[in]gntrue: Compute the Gauss-Newton Hessian, false (default): Compute the full Hessian

Compute the Jacobian matrix.

The Jacobian matrix is \(J\) such that each row is the gradient of a penalty term. The Jacobian of the \(i^{th}\) cost function makes up \(d_i\) rows of the matrix \(J\) ordered according to the ordering of \(f_i\).

This function resets the Jacobian to zero and then calls clf::CostFunction::PenaltyFunctionJacobian to compute the Jacobian matrix.

\begin{equation*} J(\beta) = \left[ \begin{array}{ccc} --- & \nabla_{\beta} f_1(\beta) & --- \\ --- & \vdots & --- \\ --- & \nabla_{\beta} f_m(\beta) & --- \\ \end{array} \right] \in \mathbb{R}^{\sum_{i=1}^{n} d_i \times n} \end{equation*}

Parameters
[in]betaThe current parameter value
[out]jacThe Jacobian matrix
