Detailed Description

Compute the coupling cost associated with a support point \(\hat{x}\) and its \(i^{th}\) nearest neighbor \(x_i\).

Let \(\Phi_{\hat{x}}(x)\) be the \(m \times \tilde{q}_{\hat{x}}\) matrix that defines the local function \(\ell_{\hat{x}}(x) = \Phi_{\hat{x}}(x) p\) for \(p \in \mathbb{R}^{\tilde{q}_{\hat{x}}}\) (see clf::SupportPoint). Similarly, let \(\Phi_{x_i}(x)\) be the \(m \times \tilde{q}_i\) matrix that defines the local function \(\ell_{x_i}(x) = \Phi_{x_i}(x) s\) for \(s \in \mathbb{R}^{\tilde{q}_i}\) associated with the neighbor point. The coupling cost function between support point \(\hat{x}\) and its \(i^{th}\) nearest neighbor \(x_i\) is

\begin{equation*} J(p, s) = \frac{c_i}{2} \| \Phi_{\hat{x}}(x_i) p - \Phi_{x_i}(x_i) s \|^2 K_i(\hat{x}, x_{i}). \end{equation*}

The input coefficeints are \([p, s]^{\top} \in \mathbb{R}^{\tilde{q}_{\hat{x}}+\tilde{q}_i}\).

Determine the local nearest neighbor index.

If \(x_i\) is one of the \(k_{nn}\) nearest neighbors of \(\hat{x}\) then this function returns the local index \(\hat{i} \in [1, k_{nn}]\) that indicates which nearest neighbor. Evaluate the Jacobian of the penalty function given the coefficients \([p, s]^{\top} \in \mathbb{R}^{\tilde{q}_{\hat{x}}+\tilde{q}_i}\).

The Jacobian of the penalty function is

Evaluate the gradient of the penalty function given the coefficients \(\beta = [p, s]^{\top} \in \mathbb{R}^{\tilde{q}_{\hat{x}}+\tilde{q}_i}\).

The gradient of the penalty function is

\begin{equation*} \frac{ \sqrt{\frac{c_i}{2} K_i(\hat{x}, x_{i})} }{ \| \Phi_{\hat{x}}(x_i) p - \Phi_{x_i}(x_i) s \| } \left[ \begin{array}{c} \Phi_{\hat{x}}(x_i)^{\top} (\Phi_{\hat{x}}(x_i) p - \Phi_{x_i}(x_i) s) \\ - \Phi_{x_i}(x_i)^{\top} (\Phi_{\hat{x}}(x_i) p - \Phi_{x_i}(x_i) s) \end{array} \right]. \end{equation*}

The input coefficeints are \(\beta = [p, s]^{\top} \in \mathbb{R}^{\tilde{q}_{\hat{x}}+\tilde{q}_i}\). Note that the gradient is zero if \(\| \Phi_{\hat{x}}(x_i) p - \Phi_{x_i}(x_i) s \| = 0\).

Parameters
[in]indThe index of the penalty function (must be \(0\) since there is only one penalty function)
Returns
The entries of the Jacobian

◆ localNeighborInd

const std::size_t clf::CoupledCost::localNeighborInd
private

The local nearest neighbor index.

The neighbor point is the \(j^{th}\) nearest neighbor of the support point associated with \(\hat{x}\).

If this is an invalid index, the cost is zero. Two points are uncoupled if they are not nearest neighbors or if they are the same point. Invalid indices are stored as the maximum possible integer (std::numeric_limits<std::size_t>::max()).

◆ scale

const double clf::CoupledCost::scale
private

The (precomputed) scale for the cost function.

The cost function for the Levenberg Marquardt algorithm takes the form \(\sum_{i=1}^{n} f_i^2\). Therefore, the constant \(c_i K_i(\hat{x}, x_i)\) actually corresponds to multiplying by \(\sqrt{c_i K_i(\hat{x}, x_i)/2}\)
