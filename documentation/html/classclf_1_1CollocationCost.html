Detailed Description

The cost function to minimize the expected least squares error with respect to a given measure.

Let \(\Phi_{\hat{x}}(x)\) be the \(m \times \tilde{q}\) matrix that defines the local function \(\ell_{\hat{x}}(x) = \Phi_{\hat{x}}(x) p_{\hat{x}}\) for \(p_{\hat{x}} \in \mathbb{R}^{\tilde{q}(\hat{x})}\) (see clf::SupportPoint). Here, \(\hat{x}\) is the nearest support point (see clf::SupportPoint). Define the approximation \(\hat{u}(x) = \Phi_{\hat{x}(x)}(x) \hat{p}_{\hat{x}(x)}\), where \(\hat{x}(x)\) is the nearest support point to \(x\).

Suppose we have \(n\) support points \(\{x_i\}_{i=1}^{n}\), we want to find \(\hat{w} \in \mathbb{R}^{n \times d}\) such that

\begin{equation*} \hat{w} = \mbox{arg min}_{w \in \mathbb{R}^{n \times d}} \int_{\Omega} \frac{1}{2} \| \mathcal{L}(\Phi_{x_{I(x)}}(x) \hat{p}_{I(x)}) - f \|^2 \pi \, dx, \end{equation*}

where \(I(x)\) is the index of the closest support point to \(x\) and

\begin{equation*} \hat{p} = \mbox{arg min}_{[n_1, ..., p_n] \in \mathbb{R}^{\tilde{q}_1} \times ... \times \mathbb{R}^{\tilde{q}_n}} \sum_{i=1}^{n} \underbrace{\sum_{j=1}^{k_i} \frac{1}{2} \|\Phi_{x_{i}}(x_j) \hat{p}_{i} - w_{j,:} \|^2 K_{x_i}(x_i, x_j) + \frac{a_i}{2} \| p_i \|}_{\text{uncoupled cost}} + \underbrace{\sum_{j=1}^{k_i} \frac{c_i}{2} \|\Phi_{x_{i}}(x_j) \hat{p}_{i} - \Phi_{x_{j}}(x_j) \hat{p}_{j} \|^2 K_{x_i}(x_i, x_j)}_{\text{Coupled cost}}. \end{equation*}

The uncoupled cost and coupled cost are computed in clf::UncoupledCost and clf::CoupledCost, respectively.
