Detailed Description

Represent a local function using a polynomial basis.

Given the number \(j \in \mathbb{N}\), define the scalar functions \(l_i: \mathbb{R} \mapsto \mathbb{R}\) such that \(l_j\) is a \(j^{th}\) degree polynomial. Let \(\iota = (i_0,\, i_1,\, ...,\, i_d)\) define a multi-index such that the \(i^{th}\) basis function \(\phi^{(i)}: \mathbb{R}^{d} \mapsto \mathbb{R}\) is

\begin{equation*} \phi^{(i(\iota))} = \prod_{j=1}^{d} l_{i_j}(x_j). \end{equation*}

Configuration Parameters:

Parameter Key Type Default Value Description
"ScalarBasis" std::string "Legendre" The orthogonal polynomial basis used for the scalar basis evaluation.

Construct a total-order polynomial basis.

For a multi-index \(\iota\) define \(\vert \iota \vert = \sum_{i=1}^{d} i_d\). This function constructs a polynomials basis using all of the multi-indices such that \(\vert \iota \vert \leq p\), where the order \(p\) is a user-supplied parameter.
