Detailed Description

Implements a model that is given to a clf::SupportPoint.

Each support point is associated with a model

\begin{equation*} \mathcal{L}(u) = f, \end{equation*}

where \(u:\Omega \mapsto \mathbb{R}^{m}\) (and \(\Omega \subseteq \mathbb{R}^{d}\)) is the function that we are trying to represent with local functions such that \(u(x) \approx \Phi_{\hat{x}}(x) p\) near \(\hat{x}\). The matrix \(\Phi_{\hat{x}}(x)\) is the matrix of basis functions associated with the clf::SupportPoint at \(\hat{x}\).

This class allows the user to implement \(\mathcal{L}\) and/or \(f\).

Configuration Parameters:

Parameter Key Type Default Value Description
"InputDimension" std::size_t 1 The input dimension \(d\).
"OutputDimension" std::size_t 1 The output dimension \(m\).
"FiniteDifferenceStep" double 1.0e-6 The parameter used to approximate derivatives with finite difference.
