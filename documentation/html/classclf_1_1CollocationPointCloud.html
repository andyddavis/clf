void CollocationPointCloud::WriteToFile ( std::string const &  filename,
std::string const &  dataset = "/" 
) const
overridevirtual

Write the points to file.

Write all of the points into a location called "collocation points". Inside this location there are data sets called "collocation points/support point i" such that each row is a collocation point associated with the \(i^{th}\) support point. If there are no collocation points assocaited with a given support point, then there is not corresponding dataset in the HDF5 file.

Parameters
filenameThe output file name (must end in ".h5")
datasetThe dataset in the hdf5 file where we store the points (defaults to root "/")
