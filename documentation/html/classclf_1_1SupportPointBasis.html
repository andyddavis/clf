Detailed Description

A wrapper around a generic basis function that scales the evaluation into "local" coordinates around a support point.

Given basis functions \(\{\hat{\phi}_i\}_{i=1}^{n}\) (e.g., clf::SinCosBasis or clf::PolynomialBasis) and a clf::SupportPoint located at \(y\) with radius \(\delta\) (see clf::SupportPoint::Radius) this class defines scaled basis functions \(\phi_i(x) = \hat{\phi}_i((x-y)/\delta)\).

Configuration Parameters:

Parameter Key Type Default Value Description
"Radius" double 1.0 The initial value of the \(\delta\) parameter.

The parameter \(\delta\) that defines the radius for which we expect this local function to be relatively accurate.

The default value is \(\delta=1\). This parameter defines the local coordinate transformation \(\hat{x}(y) = (y-x)/\delta\).
