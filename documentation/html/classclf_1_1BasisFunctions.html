Detailed Description

Evaluate the basis functions \(\phi(x) = [\phi_1(x),\, \phi_2(x),\, ...,\, \phi_q(x)]\).

We need basis functions \(\{\phi_i: \mathbb{R}^{d} \mapsto \mathbb{R} \}_{i=1}^{q}\) that define the local function \(\ell: \mathbb{R}^{d} \mapsto \mathbb{R}\). We evaluate the local function \(\ell(x) = c \cdot \phi(x)\), where \(c \in \mathbb{R}^{q}\) are basis coefficients and \(\phi(x) = [\phi_1(x),\, \phi_2(x),\, ...,\, \phi_q(x)]\).

In general, we define the basis functions by frist defining scalar functions \(l_i: \mathbb{R} \mapsto \mathbb{R}\). Let \(\iota = (i_0,\, i_1,\, ...,\, i_d)\) define a multi-index. The \(i^{th}\) basis function \(\phi_{i}: \mathbb{R}^{d} \mapsto \mathbb{R}\) is

\begin{equation*} \phi_{i(\iota)}(x) = \prod_{j=1}^{d} l_{i_j}(x_j), \end{equation*}

where we have implicitly defined a one-to-one mapping between the index \(i\) and the multi-index \(\iota\).

We assume there is one index that corresponds to a constant function \(\phi_i(x) = c\) (typically, \(c=1\)).

Evaluate the \(k^{th}\) derivative of the \(i^{th}\) basis function with respect to the \(p^{th}\) input.

Let \(\iota(i)\) be the corresponding multi-index. The basis function derivative is

\begin{equation*} \frac{d^k \phi^{(i(\iota))}(x)}{d x_p^{k}} = \frac{d^k l_{i_p}(x_p)}{d x_p^{k}} \prod_{j=1, j \neq k}^{d} l_{i_j}(x_j), \end{equation*}

where \(l_{i}\) are BasisFunctions::ScalarBasisFunction evaluations.

Evaluate the matrix of \(k^{th}\) derivatives of the basis functions with respect to the inputs.

Let \(\iota(i)\) be the corresponding multi-index. The matrix of basis function derivatives is

\begin{equation*} \mathcal{D}^{(k)} \phi(x) = \left[ \begin{array}{cccc} \frac{d^k \phi^{(1)}(x)}{d x_1^{k}} & \frac{d^k \phi^{(2)}(x)}{d x_1^{k}} & ..., & \frac{d^k \phi^{(q)}(x)}{d x_1^{k}} \\ ... & ... & ... & ... \\ \frac{d^k \phi^{(1)}(x)}{d x_d^{k}}, & \frac{d^k \phi^{(2)}(x)}{d x_d^{k}}, & ..., & \frac{d^k \phi^{(q)}(x)}{d x_d^{k}} \\ \end{array} \right] \end{equation*}
